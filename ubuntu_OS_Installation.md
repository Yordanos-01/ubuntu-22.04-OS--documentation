This project documents the installation, configuration, and system-level exploration of Ubuntu 22.04 Server in a virtualized environment using Oracle VM VirtualBox.
It includes step-by-step guidance for setting up the server, configuring networking, and managing storage during installation. 
The project also explores essential administration tools such as APT and basic Linux command-line operations.
